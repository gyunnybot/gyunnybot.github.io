## 경로 설정 시 주의할 점
윈도우 환경에서 cmd를 활용해 스프링 프로젝트를 빌드 또는 실행할 때는 영문 폴더 경로가 권장된다.

Java 런타임(JRE)은 OS의 기본 문자 셋을 따르는데, 한국의 Windows 환경은 MS949가 기본이다. Spring Project 빌드 도구인 Gradle, Maven은 결국 JVM 프로세스 위에서 돌아가기 때문에 런타임에 맞춰 MS949를 따르지만, IntelliJ와 같은 IDE나 JDK 18 이후의 javac, jar 등의 개발 툴은 UTF-8 인코딩을 기준으로 설계되었다. 따라서 Windows 환경에서 경로나 파일 이름에 한글이 포함되면 인코딩 불일치로 문제가 발생할 수 있다.

## 폴더 확인
경로를 찾았다면 명령 프롬프트(cmd)를 열고 스프링 프로젝트 폴더로 이동한 후 `dir` 명령어를 입력해 Gradle 빌드에 필요한 파일이 있는지 확인한다.

## 빌드하고 실행하기
`gradlew build` 명령어를 실행한다. BUILD SUCCESSFUL이 출력되면 /build/libs 경로로 이동해 생성된 스냅 샷의 이름을 복사 후 `java -jar XXXspring-0.0.1-SNAPSHOT.jar`와 같이 명령어를 실행하면 cmd에서 스프링 부트 화면 결과가 보이면서 서버가 실행된다.
